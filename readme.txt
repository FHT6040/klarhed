Online-kursus bygget på KLARHED-arbejdsbogen af Frank Tessin. 8 moduler. 21 udsagns baseline. 90-dages plan.

== Description ==
KLARHED er et komplet 8-moduls online-lederskabsforløb for ejerledere, topledere, iværksættere og C-level profiler.

Plugin'et indeholder:

* Baseline-måling (21 udsagn)
* 8 kapitler: Klarlægge, Lederskab, Mellemkapitel (MQ), Ambition, Resultat, Handling, Evne, Dedikation, Samling
* Refleksionsøvelser, dialogværktøjer, selvevalueringer
* Tre cases: Henriette, Martin, Lone
* 90-dages plan og personligt ledelsesmanifest
* Før/efter-sammenligning af måling

== Installation ==
1. Upload mappen `klarhed/` til `/wp-content/plugins/`
2. Aktivér plugin'et
3. Opret en side med shortcoden `[klarhed_course]`

== Shortcodes ==
* `[klarhed_course]` — hele kursusvisningen
* `[klarhed_baseline]` — kun baseline-måling
* `[klarhed_chapter slug="ambition"]` — ét kapitel
* `[klarhed_progress]` — fremskridts-widget
* `[klarhed_manifest]` — ledelsesmanifest
